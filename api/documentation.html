Track Team 67 Plants API

Endpoint type Protected Endpoint Url What Endpoing does
GET NO https://tt67-bw.herokuapp.com/ Gets the API Documentation
POST NO https://tt67-bw.herokuapp.com/users/register Registers a new user. Requires an object with a username, password, and telephone all of type string.
POST NO https://tt67-bw.herokuapp.com/users/login Logs in a user. Requires an object with a username and password all of type string. Returns JSON Web Token.
POST YES https://tt67-bw.herokuapp.com/users/plants Returns all plants associated with the logged in user.
DELETE YES https://tt67-bw.herokuapp.com/users/ Deletes currently logged in user.
POST YES https://tt67-bw.herokuapp.com/plants/ Creates a plant for logged in user. Requires a nickname, species, and h2oFrequency all of type string.
PUT YES https://tt67-bw.herokuapp.com/plants/:id Updates a plant with specified id if it belongs to logged in user. Requires what is to be updated.
DELETE YES https://tt67-bw.herokuapp.com/plants/:id Deletes the plant with specified id if it belongs to logged in user.
GET YES https://tt67-bw.herokuapp.com/plants/:id Gets plant data based of id requries logged in user to own the data.
GET NO https://tt67-bw.herokuapp.com/dummydata Dummy data for marketing page
ALL PROTECTED ROUTES REQUIRE JSON WEB TOKEN IN AUTH HEADER! YOU GET THE TOKEN IN THE RESPONSE OF THE LOGIN ENDPOINT.
